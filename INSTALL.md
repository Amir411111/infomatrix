# Инструкция по установке и запуску

## Предварительные требования

- Node.js (версия 18 или выше)
- npm или yarn
- Expo CLI (установится автоматически)
- Для iOS: Xcode (только на macOS)
- Для Android: Android Studio

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Выберите платформу:
   - Нажмите `i` для iOS симулятора (только на macOS)
   - Нажмите `a` для Android эмулятора
   - Отсканируйте QR-код в приложении Expo Go на вашем телефоне

## Структура проекта

```
clothmatch/
├── src/

│   ├── screens/          # Экраны приложения
```

## Основные функции

### Гардероб
- Добавление фото через камеру или галерею
- Выбор категории (Верх, Низ, Обувь)
- Просмотр всех вещей
- Удаление вещей

### Конструктор
- Выбор вещей из разных категорий
- Создание готового образа
- Визуальный предпросмотр

### Рекомендации
- Ввод температуры и погодных условий
- Получение AI-рекомендаций по подбору одежды

## Разрешения

При первом запуске приложение запросит разрешения на:
- Камеру (для фотографирования вещей)
- Галерею (для выбора фото)

## Troubleshooting

Если возникают проблемы:

1. Очистите кэш:
```bash
npm start -- --clear
```

2. Переустановите зависимости:
```bash
rm -rf node_modules
npm install
```

3. Для Android: убедитесь, что эмулятор запущен или устройство подключено через USB с включенной отладкой

4. Для iOS: убедитесь, что Xcode установлен и симулятор доступен
